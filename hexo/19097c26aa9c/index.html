hexo next 访问categories或tags的时候返回的是Cannot GET /tags/ 或是Cannot GET /categories/

发表于 2021-11-01 更新于 2021-11-13 分类于 hexo 阅读次数：
hexo next无法访问categories和tags

我在搭建hexo的时候，点击右侧的菜单栏里的categories的时候，返回的信息是 Cannot GET /categories/

我尝试了网络上的很多的方法
说是需要将主题配置文件（就是next的配置文件）的categories 要将“||”前的空格删除不然会导致菜单没法显示，但是尝试无效。

解决办法如下：

  1. 添加一个新页面 hexo new page "tags"
  2. 这上述命令会在 :博客路径/source/下生成一个 tags 的文件夹,打开文件夹下的 index.md,修改内容为
  3. 同理，categories也是同样的操作，
    1. hexo new page "categories"
    2. 修改 文件 :博客路径/source/categories/index.md
    3. 内容如下
      title: categories
date: 2021-11-01 10:14:08
type: categories
layout: categories
hexo Cannot GET /tags/ Cannot GET /categories/
