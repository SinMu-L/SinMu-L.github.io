php自带的内置服务

发表于 2021-11-01

最近研究读取文件内容的时候遇到这么一个命令，让我对php刮目相看
php -S localhost:8000 ./index.php

我还好奇，为什么这个命令可以生效，以为我记得php需要依赖第三方服务来运行web服务

但是好奇心的驱使下我还是运行了一下，果然给了我一个web服务，我只需要访问 http://localhost:8000 就能以web服务的形式访问这个index.php 文件

我就好奇这些命令在哪里可以寻找到相关的说明。

我第一时间就想到了官方手册，但是我直接搜索 php -S没有查询到我想要的。

我突然想到这个居然是cli命令，那么说不定可以用 -h参数来查看

然后我直接在cmd里面运行 php -h，嘿，果然找到了这个命令

我仔细看了一下这些介绍，我认为对我帮助比较大的命令也放在下面了

  3. 同理，categories 也是同样的操作，
