设置 phpstudy 显示目录结构

我最近由于某些原因需要将 localhost 下的目录结构显示出来，但是我用的集成环境 phpstudy8.1.3 ，使用的服务是 nginx ，访问 localhost 的时候就给我提示这个

这样的绝对不是我想要的，本着学习的态度询问度娘后的结果是这样子的

我按照上面的步骤设置之后重启nginx，依旧无效，依然返回403，好家伙，然后我盯着那个 0localhost_80.conf 文件瞄了几眼，瞅见了这玩意儿

好家伙，虽然我不知道这个 location 是上面作用，但是我想会不会是 location 里面是主要因素，我本着测试的态度就将 location 里面的 autoindex off; 改为 autoindex on;,然后重启nginx，再访问 localhost ，那个心仪已久的画面他来了。

后来发现，phpstudy8.1.3里面，只需要将nginx对应的配置文件里面的 location / {} 里面加上autoindex on;就可以显示目录结构了
