
随着 AI Coding 工具越来越成熟，前端开发的工作方式也正在发生变化。
如果你希望自己的项目在 **UI、UX、交互设计、组件规范、视觉层次** 等方面更加现代化，那么可以尝试一下：

[ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)

这个项目主要用于增强 AI 编程助手在前端 UI/UX 方向上的能力，让 AI 更懂现代设计语言，从而生成质量更高、更符合产品感的前端代码。

---

# 为什么值得使用？

很多 AI Coding 工具虽然能写代码，但在以下方面往往表现一般：

* UI 缺少层次感
* 组件风格不统一
* 交互体验偏“工程化”
* 缺少现代 SaaS 产品的设计感
* 页面视觉比较“朴素”

而 `ui-ux-pro-max-skill` 的目标，就是让 AI：

* 更懂现代 UI 设计
* 更懂组件布局与视觉节奏
* 更懂产品级 UX
* 更懂 Tailwind / Design System 风格
* 更容易生成高级感前端页面

尤其适合：

* SaaS 后台
* AI 产品
* Dashboard
* Landing Page
* 现代 Web App
* 移动端 H5

---

# 安装方式

首先，全局安装 CLI：

```bash
npm install -g uipro-cli
```

安装完成后，可以根据你正在使用的 AI 编程工具进行初始化。

---

# 支持的 AI 编程助手

## Claude Code

```bash
uipro init --ai claude
```

## Cursor

```bash
uipro init --ai cursor
```

## Windsurf

```bash
uipro init --ai windsurf
```

## GitHub Copilot

```bash
uipro init --ai copilot
```

## Gemini CLI

```bash
uipro init --ai gemini
```

## Codex CLI

```bash
uipro init --ai codex
```
