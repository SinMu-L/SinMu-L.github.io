一个“完整脚本”不只是写几句台词，而是一个能直接指导拍摄和剪辑的**执行文档**。你可以把它理解为：别人拿到你的脚本，不需要再问你，就能把视频拍出来。

接下来从**结构 + 内容 + 专业版本**三个层级讲清楚。

---

# 一、最基础结构（必须有的 6 个部分）

一个完整脚本，至少包含：

## 1. 基本信息（项目层）

说明这条视频是什么：

* 标题 / 主题
* 时长（比如30秒 / 3分钟）
* 风格（搞笑 / 高级 / 真实）
* 平台（抖音 / B站 / YouTube）

👉 作用：统一方向，避免拍偏

---

## 2. 故事结构（内容骨架）

也就是视频的“起承转合”：

常见结构：

* 开头（Hook）：吸引注意
* 中间（内容展开）
* 结尾（总结 / 反转 / 引导）

👉 作用：决定观众会不会看完

---

## 3. 场景划分（Scene）

把内容按时间/地点拆开：

例子：

* 场景1：办公室
* 场景2：地铁
* 场景3：家

👉 每个场景通常包含多个镜头

---

## 4. 镜头设计（Shot List）

每个场景里要写清楚：

* 景别（远景 / 中景 / 特写）
* 运镜（推 / 拉 / 跟 / 手持）
* 画面内容（人物在干嘛）

👉 作用：让摄影知道怎么拍

---

## 5. 语言内容（话术 + 旁白）

包括：

### 话术（人物说的话）

* 对话
* 口播

### 旁白（画外音）

* 解释信息
* 情绪表达

👉 作用：传递信息 + 情绪

---

## 6. 后期提示（剪辑/音效/字幕）

很多人会忽略，但其实很关键：

* 哪段要快剪 / 慢放
* 是否加音乐
* 字幕重点
* 转场方式

👉 作用：决定“成片好不好看”

---

# 二、标准脚本格式（最常用）

实际工作中，常见是这种“表格型脚本”：

| 场景  | 镜头 | 画面   | 话术       | 旁白       | 时长 | 备注  |
| --- | -- | ---- | -------- | -------- | -- | --- |
| 办公室 | 特写 | 打卡下班 | “终于下班了”  | 无        | 3s | 音乐起 |
| 地铁  | 中景 | 人群拥挤 | 无        | “城市开始降温” | 5s | 慢镜头 |
| 家里  | 近景 | 吃饭   | “还是家最舒服” | 无        | 4s | 暖色调 |

👉 这个就是“导演 + 摄影 + 剪辑”都能用的版本

---

# 三、进阶：专业脚本会多哪些东西

如果更专业一点（广告 / 剧情 /团队拍摄），还会增加：

## 1. 分镜编号（Shot ID）

* 镜头1 / 镜头2
  👉 方便沟通和修改

---

## 2. 构图说明

* 人物在左 / 居中 / 对称
  👉 提高画面美感

---

## 3. 情绪 / 表演提示

* 开心 / 疲惫 / 压抑
  👉 指导演员

---

## 4. 道具 / 服装

* 咖啡杯 / 西装 / 手机
  👉 避免现场混乱

---

## 5. 音乐 / 音效（BGM / SFX）

* 哪一秒进入
  👉 强化节奏和情绪

---

## 6. 转场设计

* 切换 / 淡入 / match cut
  👉 让视频更流畅

---

# 四、不同类型视频的“脚本侧重点”

## 1. 短视频（抖音/小红书）

重点：

* 开头3秒（极重要）
* 话术（抓人）
* 节奏（快）

👉 镜头可以简单，但结构必须强

---

## 2. Vlog

重点：

* 场景真实
* 旁白自然

👉 脚本可以“半结构化”

---

## 3. 广告 / 品牌片

重点：

* 分镜精细
* 风格统一

👉 脚本非常详细

---

## 4. 剧情片

重点：

* 人物关系
* 情节推进

👉 更像“电影剧本”

---

# 五、一句话总结

👉 **完整脚本 = 内容结构 + 画面设计 + 语言表达 + 执行说明**

或者更直白一点：

👉 **它要同时回答四个问题：**

* 拍什么（脚本/场景）
* 怎么拍（镜头）
* 说什么（话术/旁白）
* 怎么变好看（剪辑/音乐/节奏）

---


脚本提示词通用版
```
你是一名专业视频导演和编剧，请根据用户输入，生成一个“可直接拍摄执行”的完整视频脚本。

【用户输入】
主题：{主题}
类型：{短视频 / vlog / 广告 / 剧情}
时长：{例如30秒 / 1分钟}
风格：{搞笑 / 高级感 / 真实 / 情绪 / 种草等}
目标平台：{抖音 / 小红书 / B站 / YouTube}
目标受众：{可选}
核心目的：{涨粉 / 带货 / 表达观点 / 品牌传播等}

【要求】

1. 输出必须结构清晰，包含以下部分：

【一、整体创意】
- 一句话概念（这个视频的核心亮点）
- 情绪基调
- 节奏（快 / 中 / 慢）

【二、故事结构】
- 开头（Hook：3秒内吸引人）
- 中间（内容展开）
- 结尾（反转 / 升华 / 引导互动）

【三、分场景脚本（重点）】
请用表格形式输出，每一行包含：

| 场景 | 镜头 | 画面描述 | 话术 | 旁白 | 时长 | 拍摄/剪辑备注 |

要求：
- 镜头要具体（如：特写 / 跟拍 / 手持）
- 画面要可执行（人物在做什么）
- 话术要自然、有传播力
- 旁白要有情绪或信息增量

【四、风格与执行建议】
- 画面风格（色调 / 构图）
- 剪辑节奏
- 音乐建议
- 字幕建议

【五、可优化方向（改写建议）】
- 如何增强前3秒吸引力
- 是否有更强的情绪点或传播点

【额外要求】
- 输出要具体，不要抽象描述
- 避免空话（如“很好看”“很有趣”）
- 让一个没有经验的人也能照着拍
```
